# VSPP(Verbalizing sensibilities about piano pieces)

## サイト概要

ピアノ楽曲を聴いた感想を投稿し、共有できるSNSです。フォロー・フォロワー機能では、例えば自分が好きな曲に良いコメントを残していた会員をフォローして、
他にどのような曲に対してコメントを残しているのかをチェックできます。
リスト機能では、自分のお気に入り曲として管理するだけではなく、後で聴いてみたい曲、今後弾いてみたい曲としてまとめる等、
会員独自の基準で作成可能です。検索機能では、楽曲検索、作曲家検索ができます。インクリメンタル検索で、うろ覚えでの検索にも対応しています。
検索結果にデータがある場合には、感想を追記することができます。しかし、楽曲数はたくさんあるため、管理者側で楽曲データを追加する作業は大変です。
そのため、検索結果に楽曲が表示されなかった場合には、会員側は楽曲名と作曲家名の入力を必須とし、新規追加の申請ができます。
その後、管理者が申請内容を確認・承認し、その該当データが会員側にも表示され、感想を追記することができます。
会員は楽曲名や作曲家目を新規追加できるため、実際には存在しない楽曲名等を記入してしまう可能性を考慮し、
管理者側は楽曲名の編集及び削除をできるようにしています。

### サイトテーマ

ピアノ楽曲の感想を投稿して共有し、楽曲をリスト管理できるSNS

### テーマを選んだ理由

匿名のスレッド形式で楽曲をオススメされても、その人の好みもわからず、なかなか聴こうとは思いづらいですが、
フォロー・フォロワー機能を活用することで、「この人がこういう曲が好きなら、ぜひ私も一度聴いてみよう」と思えます。
楽曲名で検索すると、個人ブログで感想が執筆されていたりします。ただそれは一個人の意見であって、そこから発展はありませんが、当サイトでは感想の投稿と共有によって、より深く楽曲を理解することができます。
また、Youtubeにはコメント機能があり、その動画ページに遷移していないと、その楽曲に対するコメントを見ることができませんが、当サイトではコメントから判断して楽曲を聴いてみることができます。
音楽は、同じ楽曲であっても自分の体調などの状態や置かれた環境によって感じ方が異なることがあります。そのため、「この音楽、好きだな」と思った時に、感想を記録しておくことで、
自分の感性を磨くことができます。そして「この音楽、少し苦手だな」、「この音楽の良さ、よくわからないな」という時にも感想を記録しておくことで、
時間を空けて聴いた際には新たな発見ができるかもしれません。
感想を投稿することで自分の感性を言語化し、相手への伝達力を鍛えることができると考えます。
単に「素晴らしい」「カッコいい」という言葉で終わらせることなく、より深く言語化しようと心がけることで、
それに伴いピアノ楽曲を意識的に聴くようになると思います。受動的に聴く音楽も良いですが、自分の気分にあった楽曲を探し出したり、
有名ではない楽曲も聴いてみようと思えることが大切だと思っています。

### ターゲットユーザ

ピアノ楽曲を愛してやまない音楽愛好家、好きなピアノ楽曲を新たに発見したい方

### 主な利用シーン

Youtube などの動画サイトを開いている時、コンサートの帰り道、CDなどを聴いている時
