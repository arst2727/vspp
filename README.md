# VSPP(Verbalizing sensibilities about piano pieces)

## サイト概要

ピアノ楽曲を聴いた感想を投稿し、共有できるSNSです。<br>
利用者一覧では各会員の投稿数やサイト登録日などが確認できます。<br>
自分の好きな楽曲にコメントを残していた会員のページを見にいくと、<br>
他にどのような曲に対してコメントを残しているのか等をチェックできます。<br>
各会員のページではカレンダーにその日何件投稿したかが記録されます。<br>
会員をフォローすると、自分のタイムラインにフォロワー全員のコメントが最新順で表示されます。<br>
リスト機能では、自分のお気に入り楽曲として管理するだけではなく、<br>
後で聴いてみたい曲、今後弾いてみたい曲としてまとめる等、会員独自の基準で複数作成可能です。<br>
検索機能では、楽曲検索、作曲家検索ができます。<br>
検索ボタンを押す前に候補を表示させることで、うろ覚えでの検索にも対応しています。<br>
検索結果にデータがある場合には、感想を追記することができます。<br>
しかし、楽曲数はたくさんあるため、管理者側で楽曲データを追加する作業は大変です。<br>
そのため、検索結果に楽曲が表示されなかった場合には、<br>
会員側は楽曲名と作曲家名の入力を必須とし、新規追加の申請ができます。<br>
その後、管理者が申請内容を確認・承認し、その該当データが<br>
会員側にも表示され、感想を追記することができます。<br>
会員は楽曲名を新規追加できるため、実際には存在しない楽曲名等を記入してしまう可能性を考慮し、<br>
管理者側は楽曲名の編集及び削除をできるようにしています。

### サイトテーマ

ピアノ楽曲の感想を投稿して共有し、楽曲をリスト管理できるSNS

### テーマを選んだ理由

匿名のスレッド形式で楽曲をオススメされても、その人の好みもわからず、なかなか聴こうとは思いづらいですが、<br>
フォロー・フォロワー機能を活用することで、「ラフマニノフ好きの〇〇さんがこの曲にコメントを<br>
残しているから、私も一度聴いてみよう」と思えます。<br>
楽曲名で検索すると、個人ブログで感想が執筆されていたりします。<br>
しかし共有という点で、基本的にはもともと楽曲名を知っている人のみターゲットとなります。<br>
当サイトでは感想の投稿と共有によって、より深く楽曲を理解することができます。<br>
また、Youtubeにはコメント機能があり、その楽曲や演奏自体に対するコメントを見られますが、<br>
これも共有という点で楽曲名をあらかじめ知っている必要があります。<br>
当サイトでは、楽曲名を知らなくてもコメントから判断して楽曲を聴いてみることができます。<br>
音楽は、同じ楽曲であっても自分の体調などの状態や置かれた環境によって感じ方が異なることがあります。<br>
そのため、「この音楽、好きだな」と思った時に、感想を記録しておくことで、<br>
自分の感性を磨くことができます。そして「この音楽、少し苦手だな」、<br>
「この音楽の良さ、よくわからないな」という時にも感想を記録しておくことで、<br>
時間を空けて聴いた際には新たな発見ができるかもしれません。<br>
感想を投稿することで自分の感性を言語化し、さらには他者への伝達力を鍛えることができると考えます。<br>
単に「素晴らしい」「カッコいい」という言葉で終わらせることなく、より深く言語化しようと心がけることで、<br>
それに伴いピアノ楽曲を意識的に聴くようになると思います。<br>
受動的に聴く音楽も良いですが、自分の気分にあった楽曲を探し出したり、<br>
有名ではない楽曲も聴いてみようと思えることが大切だと思っています。

### ターゲットユーザ

ピアノ楽曲を愛してやまない音楽愛好家、好きなピアノ楽曲を新たに発見したい方

### 主な利用シーン

Youtube などの動画サイトを開いている時、コンサートの帰り道、CDなどを聴いている時
